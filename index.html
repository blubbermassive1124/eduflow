首次設定 — 只需 3 分鐘
① 填入 Supabase 設定
📋 取得步驟：
1. 前往 supabase.com → 免費建立帳號
2. 點擊「New Project」→ 輸入名稱 → 設定密碼 → 建立
3. 等待建立完成 → 點擊「Settings」→「API」
4. 複製「Project URL」和「anon public」key 填入下方
② 填入 Claude API Key（AI 出題功能）
📋 取得步驟：
1. 前往 console.anthropic.com
2. 點擊「Create Key」→ 輸入名稱
3. 複製以 sk-ant- 開頭的 Key 填入下方
4. 確認帳戶有餘額（新帳號有免費額度）
③ 建立資料庫（複製 SQL 執行）
📋 步驟：
1. 回到 Supabase 後台 → 左側選「SQL Editor」
2. 點擊「New query」
3. 把下方所有 SQL 貼入 → 點擊右上角「Run」
4. 看到「Success」後回來點「完成設定」
